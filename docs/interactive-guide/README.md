# OpenClaw 인터랙티브 학습 가이드

초등학생도 이해할 수 있는 OpenClaw 아키텍처 시각 학습 가이드입니다.

## 파일 구조

| 파일 | 내용 | 레슨 |
|------|------|------|
| `index.html` | 메인 지도 | 1: 시스템 구조, 4: 성장 이야기, 5: 확장 생태계, 9: 설계 원칙 |
| `message-flow.html` | 흐름 | 2: 메시지 흐름, 6: 훅 생명주기, 7: 에이전트와 세션 |
| `systems.html` | 시스템 | 3: 플러그인 시스템, 8: 설정 파이프라인 |

## 사용법

HTML 파일을 브라우저에서 직접 열면 됩니다. 서버 불필요.

```bash
# 또는
open docs/interactive-guide/index.html
```

## 특징

- 모든 설명이 한국어 + 실생활 비유 (우체국, 식당, 지하철, 레고 등)
- 클릭하면 상세 설명이 나오는 인터랙티브 SVG 다이어그램
- 메시지 흐름 애니메이션
- 미니 퀴즈
- 외부 의존성 없음 (Pretendard 폰트 CDN만 사용)
- 모바일 반응형
